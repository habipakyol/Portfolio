## Overview

Development of a scalable and user-friendly note-sharing platform for students of Mimar Sinan Fine Arts University – Mathematics Department, built with ASP.NET Core 7.0 MVC. The project integrates Microsoft Azure AD for secure authentication, utilizes Entity Framework Core with SQL Server for efficient data management, and features a modern, responsive UI designed with Bootstrap 5.

## Key Features

- Secure Institutional Login: Integrated Microsoft Azure AD to enable secure authentication using university emails (ogr.msgsu.edu.tr), with role-based authorization for admins.
- Course-Based PDF Organization: Students can upload and access lecture notes categorized by course, ensuring structured and easy-to-navigate content.
- Favorites System: Users can bookmark frequently accessed documents for quick access.
- User Profiles: Each student has a dedicated profile page displaying their uploaded and favorite documents.
- Admin Dashboard: Built a secure admin panel to manage courses, documents, and user permissions efficiently.

## Technologies Used

- ASP.NET Core 7.0 MVC: For building a robust and scalable server-side application structure.
- Microsoft Azure AD: For secure single sign-on (SSO) and user identity management.
- Entity Framework Core: To handle ORM-based data access and management.
- SQL Server: As the primary database for storing course data, documents, and user preferences.
- Bootstrap 5: For developing a modern, responsive, and mobile-friendly UI.

## Admin Capabilities

- Course Management: Add, update, or remove course entries with full CRUD functionality.
- Document Control: Moderate and manage all uploaded documents.
- User Access Management: Assign roles and control access levels with integrated role-based permissions.


## Core Functionalities

- PDF Upload & Download: Easily upload lecture notes in PDF format and download them across devices.
- Favorites and Organization: Organize documents by course and mark important files as favorites.
- Profile Integration: View personal uploads and bookmarked documents via user profile pages.

## Authentication and Authorization

- Azure AD Login: Enforced institutional email validation to restrict access to authorized students.
- Admin Roles: Role-based access control with elevated permissions for administrators.

## Database Structure

- Courses table for managing class metadata.
- PdfDocuments table for storing uploaded files.
- FavoriteDocuments table to handle user-specific favorites.

## User Experience (UX) Enhancements

- Responsive Design: Ensures optimal viewing across devices using Bootstrap 5.
- Animated Notifications: Visual cues for uploads, errors, and actions improve interactivity.
- Clean Navigation: Intuitive UI and clear call-to-actions for smooth user interaction.

## Outcome
The platform offers a streamlined solution for digital note sharing within MSGSÜ Mathematics Department. It improves collaboration among students, ensures secure access via institutional accounts, and provides admins with full control over educational content — all wrapped in a responsive and user-friendly interface.
